strings库学习笔记

strings库包括一些用于以UTF-8编码的字符串的处理方法。内容包括：

  • 类型：

    // 用于通过Write方法高效建立string。最小化了内存复制。零值可用。不要复制非空Builder
// type Builder struct {
// 	addr *Builder // of receiver, to detect copies by value
// 	buf  []byte
// }
type Builder

    // 用于从string中读取。零值相当于空string的Reader
// type Reader struct {
//	s        string
//	i        int64 // current reading index
//	prevRune int   // index of previous rune; or < 0
// }
type Reader

    // 用于批量进行string的替换动作。go程安全
// type Replacer struct {
//	once   sync.Once // guards buildOnce method
//	r      replacer
//	oldnew []string
// }
type Replacer

  • 函数：

    • 比较：Compare、EqualFold；
    • 包含：Contains、ContainsAny、ContainsRune、HasPrefix、HasSuffix、
    • 统计：Count；
    • 拆分：Fields、FieldsFunc、Split、SplitAfter、SplitAfterN、SplitN；
    • 索引：Index、IndexAny、IndexRune、IndexFunc、LastIndex、LastIndexAny、LastIndexByte、LastIndexFunc；
    • 合并：Join；
    • 变更：Map、Repeat、Replace、ReplaceAll、ToLower、ToLowerSpecial、ToTitle、ToTitleSpecial、ToUpper、ToUpperSpecial、ToValidUTF8；
    • Trim：Trim、TrimFunc、TrimLeft、TrimLeftFunc、TrimRight、TrimRightFunc、TrimPrefix、TrimSuffix、TrimSpace；

  • Builer函数：

    • 属性：Cap、Grow、Len；
    • 删除：Reset；
    • 转换：String；
    • 写：Write、WriteByte、WriteRune、WriteString；

  • Reader函数：

    • 生成：NewReader；
    • 属性：Len，Size；
    • 删除：Reset；
    • 查找：Seek；
    • 撤回：UnreadByte、UnreadRune；
    • 读：Read、ReadAt、ReadByte、ReadRune；
    • 写：WriteTo；

  • Replacer函数：

    • 生成：NewReplacer；
    • 执行：Replace、WriteString；
