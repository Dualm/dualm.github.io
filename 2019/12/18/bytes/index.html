bytes库学习笔记

bytes库主要用于byte类切片的处理。包括内容如下：

  • 常量：

    const MinRead = 512

  • 变量：

    var ErrTooLarge = errors.New("bytes.Buffer: too large")

  • 类型：

    • // 大小可变的bytes数据缓冲，拥有Read和Write方法。零值为空Buffer
// type Buffer struct {
//  buf      []byte // contents are the bytes buf[off : len(buf)]
//  off      int    // read at &buf[off], write at &buf[len(buf)]
//  lastRead readOp // last read operation, so that Unread* can work correctly.
// }
type Buffer

    • // 增加了Read、Seek方法的byte切片，并且是只读的。零值为操作空切片的Reader
// type Reader struct {
//  s        []byte
//  i        int64 // current reading index
//  prevRune int   // index of previous rune; or < 0
// }
type Reader

  • 函数：

    • byte切片处理：

    • 比较：Compare、Equal、EqualFold；

    • 包含：Contains、ContainsAny、ContainsRune、HasPrefix、HasSuffix；

    • 统计：Count；

    • 拆分：Fields、FieldsFunc、Split、SplitAfter、SplitAfterN、SplitN；

    • 合并：Join；

    • 变更元素：Map、Repeat、Replace、ReplaceAll、Title、ToLower、ToLowerSpecial、ToTitle、ToTitleSpectial、ToUpper、ToUpperSpecial、ToValidUTF8；

    • 获取元素索引：Index、IndexAny、IndexByte、IndexFunc、IndexRune、LastIndex、LastIndexAny、LastIndexByte、LastIndexFunc；

    • 类型转换：Rune、

    • Trim：Trim、TrimFunc、TrimLeft、TrimLeftFunc、TrimRight、TrimRightFunc、TrimPrefix、TrimSuffix、TrimSpace；

    • Buffer：

    • 生成：NewBuffer、NewBufferString；

    • 类型转换：Bytes、String；

    • 属性：Cap、Grow、Len；

    • 读：Next、Read、ReadByte、ReadBytes、ReadFrom、ReadRune、ReadString；

    • 重置读：UnreadByte、UnreadRune；

    • 删除：Reset、Truncate；

    • 写：Write、WriteByte、WriteRune、WriteString、WriteTo；

    • Reader：

    • 生成：NewReader；

    • 属性：Len、Size；
